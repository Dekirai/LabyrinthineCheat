# Labyrinthine Cheat
 An ImGUI cheat for Labyrinthine using MelonLoader 7.0

Tools used: dnSpy, Visual Studio, .net6, Il2CppDumper

Tutorial: https://www.youtube.com/watch?v=XH4lePNqiHc

### Functions:
In Lobby
- Unlock all cosmetics
- Unlock all monster types
- Unlock all maze types
- Have all items x1000
- Give Hardcore and Rare Stamps
- Give/Remove currency
- Give/Remove Experience
  
In-Game
- Complete Case (Also picks up any cosmetic if one exists)
- Disable/Enable Monsters
- Self Revive
- Toggle ESP (Monster & Player)
- Set Flashlight Power
- Set movement speed modifier
- Set jump height modifier
- Teleport to Monster
- Teleport to Spawnpoint
- Teleport to custom coordinates
- Teleport to Safezones
